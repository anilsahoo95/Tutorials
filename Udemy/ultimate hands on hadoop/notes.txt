hadoop is a open software platform for distributed storage and distributed processing of large amount of dataset on clusters of computer

HDFS- hadoop distributed file system
stores huge sets of data across machines.. also enables redundancy so that if one machine goes down data is saved
mapreduce is used to process large amount of data across machines
