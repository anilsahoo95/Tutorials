data

order,meaning and value

Coding,Maths and domain knowledge..

1.Statistics- Python and R
2.database- SQL
3.Command Line- Bash
4.Search-- Regex-- find data 

Maths- probability,algebra and regression

maths is needed to choose a procedure to sol
ve a problem

1.Planning-- define the goals of the project
2.Data Prep-- get,clean,explore and refine 
data
3.Modelling-- create and validate the model
